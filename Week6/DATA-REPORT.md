### DATA REPORT

We downloaded a raw dataset of Border Crossing from `data.gov` in `.csv` format.

### DATASET DESCRIPTION

Data provides summary statistics for inbound crossings at the US-Canada and the US-Mexico broder at the port level. Data is available for trucks, trains, containers, buses, personal vehicles, passengers, and pedestrians. The data reflects the number of vehicles, containers, passengers or pedestrians entering the US.

We have divided the the dataset into:

- Port Name
- State
- Port Code
- Border
- Date
- Measure
- Value
- Lattitude
- Longitude
- Point

The total no of entries in the dataset are **_406721_**.
